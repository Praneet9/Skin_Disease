### Requirements
- Tensorflow-gpu 1.13.0
- Keras 2.2.4
- OpenCV 4.0
- Numpy 1.16.1
- Python-telegram-bot 11.1.0
- Flask

### Run telegram bot
```
python scripts/remedium_bot.py
```

### Launch Flask Server for API
```
python server.py
```
